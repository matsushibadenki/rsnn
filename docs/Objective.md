**目標**

① 認識精度が95〜97%（CIFAR-10換算）に到達し、
② 消費電力がANN比1/50以下で、
③ 学習再現性が95%以上に安定すること。
④ 行列計算やGPUに頼らないこと。
⑤ 誤差逆伝播学習法を使わない方法を模索すること。


🧠 1. 主要な精度の評価軸（到達すべき「及第点」）

SNNがANNの代替・上位互換として認められるには、以下の5つの精度領域でANNと同等以上になる必要があります。
| 分野                                       | 精度の意味                 | 現状のSNN精度                     | ANNとの比較                       |
| ---------------------------------------- | --------------------- | ---------------------------- | ----------------------------- |
| **(1) 認識精度（分類タスク）**                      | 画像・音声・時系列認識の正答率       | CIFAR-10で90〜94%（ResNet-SNN系） | ANN(ResNet): 95〜97% → あと1〜2%差 |
| **(2) 時系列予測精度（Temporal prediction）**     | スパイク時系列から未来の状態を推定する能力 | 短期では良好だが長期予測で誤差累積            | ANN(LSTM, Transformer)に劣る     |
| **(3) ノイズ耐性精度**                          | 入力ノイズ・信号劣化下での性能       | SNNは本質的に高い（神経的冗長性）           | ここではSNNが有利                    |
| **(4) エネルギー効率 vs 精度トレードオフ**              | 同じ電力量で達成できる精度         | ANNより10〜100倍効率的だが精度低下あり      | 「効率×精度」積で拮抗点を目指す              |
| **(5) 学習再現性・安定性（Convergence stability）** | 同じデータ・設定で再学習しても結果が安定  | SNNはまだ収束が不安定（スパイク揺らぎ）        | ANNでは収束再現性が高い                 |


🧩 2. ANNに追いつくための「ボトルネック」と必要条件

| 領域        | ボトルネック                                  | 改善すべき「精度」要素                               |
| --------- | --------------------------------------- | ----------------------------------------- |
| **学習段階**  | 勾配非連続（spikeの離散性）                        | 「Surrogate Gradient（擬似微分）」の近似精度を高める必要     |
| **伝達段階**  | スパイク符号化（rate coding・temporal coding）の誤差 | コーディング精度（temporal precision < 1ms）を保証する必要 |
| **実装段階**  | ニューロモルフィックハードウェアの量子化誤差                  | 実機のスパイク時刻誤差を±1ms以内に制御すること                 |
| **モデル構造** | ANN変換型SNNでは時間次元の情報が損失                   | SNNネイティブ構造（時間自己回帰構造）で精度を稼ぐ                |


📊 3. 数値目標としての「及第点ライン」

| 評価項目                                     | 現在の平均          | 及第点（実用ライン） | 備考                |
| ---------------------------------------- | -------------- | ---------- | ----------------- |
| 画像認識精度（CIFAR-10）                         | 90〜94%         | ≥96%       | ANNと同等レベル         |
| エネルギー効率（消費電力）                            | ANN比1/20〜1/100 | ≥1/50で同精度  | 実用デバイスで優位性確保      |
| 学習安定率（再現性）                               | 約70〜80%        | ≥95%       | 再学習でも安定再現する必要     |
| 時系列認識（Spiking Speech, EEG, Event camera） | 85〜88%         | ≥92%       | ANN並みにリアルタイム動作可能に |
| 推論レイテンシ（ms）                              | 数十ms           | ≤10ms      | リアルタイム応答条件（ロボット等） |
